## OPEN-API

> JavaScript base website script using a plugin as a router that is lightweight and easy to customize.

### Features

- [x] Plugin as Router
- [x] Auto List Feature
- [x] High Optimation
- [x] Easy to Customize
- [x] Suitable for Rest API
- [x] etc.

### Installation & Run

```
$ yarn
$ node .
```

or want to use pm2

```
$ yarn
$ npm i -g pm2
$ pm2 start index.js && pm2 save && pm2 logs
```
